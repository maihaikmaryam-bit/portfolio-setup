# Portfolio Project Setup

## Tools Installed
- Cursor IDE
- Claude Code Add-on (by Anthropic)
- Codex Add-on (by OpenAI)

## Steps Completed
1. Downloaded and installed Cursor IDE on Windows.
2. Navigated to the Extensions marketplace and successfully installed the Claude Code and Codex extensions.
3. Created a public repository on GitHub.
4. Installed Git for Windows locally to resolve command terminal pathing.
5. Successfully cloned the repository into Cursor and created this documentation.

## Issues and Solutions
- *Issue:* Microsoft Defender SmartScreen blocked the Cursor installer.
*Solution:* Bypassed it safely by clicking "More Info" and selecting "Run Anyway".
- *Issue:* Terminal threw a 'git is not recognized' error during cloning.
*Solution:* Downloaded and installed Git for Windows, restarted the Cursor IDE, and ran the clone command successfully.
